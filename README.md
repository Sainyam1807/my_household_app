## Overview
HomeEasy is a full-stack household services marketplace that connects customers with verified service professionals. The platform enables end-to-end service booking, request tracking, and feedback management through a role-based workflow system.

---

## Key Features
- Service booking and request lifecycle management
- Role-based access for Customers, Professionals, and Admins
- Search and approval workflows for service professionals
- User profiles and service history tracking
- Analytics dashboards with interactive charts
- Feedback and rating system

---

## Tech Stack
- **Backend:** Flask
- **ORM:** Flask-SQLAlchemy
- **Authentication:** Flask-Security
- **Frontend:** Jinja2 Templates, HTML, Bootstrap
- **Database:** SQLite
- **Visualization:** Chart.js
- **APIs:** RESTful APIs for client-server communication

---

## Roles
- **Customer:** Book services, track requests, and provide feedback  
- **Professional:** Manage service requests and profile  
- **Admin:** Approve professionals, manage users, and monitor platform analytics  

---
## Project Type
Full-stack web application for service marketplace management.
